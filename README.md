# Student-Management-System
A simple Python-based Student Management System that demonstrates Object-Oriented Programming (OOP) concepts and basic CRUD (Create, Read, Update, Delete) operations. This project allows users to manage student records through a menu-driven console interface.

##  Features

-  Add new student records
-  View all students
-  Search student by ID
-  Update student marks
-  Delete student records
-  Display student with highest marks
-  User-friendly menu-driven interface

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/StudentMangementSystem.git
```

2. Navigate to the project directory:

```bash
cd StudentMangementSystem
```

3. Run the program:

```bash
python student_management_system.py
```

##  Usage

When you run the program, you will see the following menu:

```text
===== Student Management System =====

1. Add New Student
2. View All Students
3. Search Student by ID
4. Update Student Marks
5. Delete Student Record
6. Display Student with Highest Marks
7. Exit
```

### Example

```text
Enter Student ID: 101
Enter Student Name: John
Enter Age: 20
Enter Marks: 89

Student added successfully!
```

##  Functionalities

### Add Student
Stores:
- Student ID
- Name
- Age
- Marks

### View Students
Displays all student records available in the system.

### Search Student
Finds a student using their unique Student ID.

### Update Marks
Updates marks for an existing student.

### Delete Student
Removes a student record from the system.

### Highest Marks Student
Displays the student who scored the highest marks.
